Run this script by using generate.py

This needs java installed and working if custom javaBinary use the --javapath argument

The curseforgeCli app is made by North-West-Wind: https://github.com/North-West-Wind/CurseForge-CLI

To install put "curseforge-cli.jar" from github inside the "North-West-Wind_CurseForge-CLI" folder, then run generate.py
